## 1. 项目概述

### 1.1 项目目标

构建玻璃杯专用 AI 个性化生成 + 多目标工艺优化系统：

1. 用户输入中文设计需求（如"简约圆柱形透明玻璃杯"）
2. 后端调用 Stable Diffusion WebUI 生成 4 张产品级玻璃杯设计图
3. 对选中图片执行 MOPSO 多目标粒子群优化，求解最优成型工艺参数
4. 输出完整工业设计档案：效果图 + 加热温度/吹气压力/冷却速率等全套工艺参数

## 3. 算法实现详解

### 3.1 MOPSO 多目标粒子群优化

**算法选型**：不使用 NSGA-II 或 pymoo，**纯自实现 MOPSO**。

**核心参数**：

| 参数 | 值 | 理由 |
|------|----|------|
| 粒子数 | 50 | 6 维搜索空间，50 粒子足够覆盖 |
| 最大迭代 | 100 | >100 代后收敛曲线趋于平坦 |
| 惯性权重 w | 0.5 | 固定值，平衡探索/开发 |
| 个体学习因子 c1 | 1.5 | 标准 PSO 推荐值 |
| 社会学习因子 c2 | 1.5 | 标准 PSO 推荐值 |
| 最大归档数 | 100 | 保持帕累托前沿多样性 |
| 速度限幅 | 边界跨度 10% / 步 | 防止粒子飞出边界 |

**算法流程**：

```
初始化 50 个粒子（位置在边界内均匀随机，速度 [-1,1]）
↓
for 代 = 1 to 100:
  评估每个粒子的 4 目标适应度
  更新个体最优（如果新位置支配旧个体最优）
  更新帕累托归档（非支配解筛选 + 去重 + 均匀抽稀）
  for 每个粒子:
    从帕累托前沿随机选一个解作为全局引导（保持多样性）
    更新速度: v = w*v + c1*r1*(pBest - x) + c2*r2*(gBest - x)
    速度限幅
    更新位置: x = x + v
    位置边界约束
  记录本轮历史（每代均值 + 最优值）
↓
输出帕累托前沿
归一化加权选最优解（先归一化再等权加权，消除量纲差异）
```

**关键实现特点**：

| 特点 | 说明 |
|------|------|
| **纯自实现** | 不依赖 DEAP/pymoo，全部手写 |
| **均匀抽稀** | 归档超限时按首目标排序均匀抽样，非拥挤距离 |
| **随机全局引导** | 每代从帕累托前沿随机选解作为 gBest，防止陷入局部最优 |
| **4 目标全部最小化** | weights 一律 -1.0 |
| **v2.4 归一化加权** | 选最优解前先归一化（min-max），防能耗（~5000-14000）主导其他目标（~0.01-0.40） |
| **v2.4 量产可行性PI** | PI 指数（0-1）+ 5 级标签（非常易量产/易量产/可量产/较难量产/不建议量产），权重：均匀度 0.30 / 废品率 0.35 / 能耗 0.15 / 耐热 0.20 |

### 3.2 目标函数（工艺数学模型，含轮廓复杂度 C 联动）

> ⚠️ **重要提醒**：以下公式均为**经验公式**，系数（0.08/0.04/0.1/100/5 等）**未经过真实生产数据回归**。变量范围（550-700°C 等）取自真实工艺手册量级。改进方向：DOE 实测 + 多元线性回归校准。

**决策变量（6 维）**：

| 变量 | 范围 | 单位 | 含义 |
|------|------|------|------|
| heating_temp | [550, 700] | °C | 加热温度 |
| heating_time | [30, 180] | 秒 | 加热时间 |
| blowing_pressure | [0.1, 0.5] | MPa | 吹气压力 |
| blowing_time | [5, 30] | 秒 | 吹制时间 |
| cooling_rate | [10, 50] | °C/min | 冷却速率 |
| wall_thickness_target | [2, 5] | mm | 目标壁厚 |

**轮廓复杂度 C（0~1）**：由 [feature_extractor.py](file:///d:/trae/玻璃杯AI生成可视化设计系统/src/services/feature_extractor.py) 对生成图进行 Canny 边缘检测 + 轮廓分析后提取，自动传入目标函数。

**目标 1：壁厚均匀度（最小化 CV）**

```python
def calculate_wall_uniformity(params):
    complexity = params.get("complexity", 0.5)
    temp_factor = (params["heating_temp"] - 550) / 150           # [0, 1]
    pressure_factor = 1 - abs(params["blowing_pressure"] - 0.3) / 0.2
    base_std = 0.12 + complexity * 0.06                          # C↑ → 更难均匀
    uniformity_gain = (1 - complexity * 0.4) * (temp_factor * 0.08 + pressure_factor * 0.04)
    std_dev = max(0.02, base_std - uniformity_gain)
    cv = std_dev / params["wall_thickness_target"]
    return cv
```

- C 越高 → 基础标准差增大（复杂形状更难均匀），温度和压力的均匀化收益降低

**目标 2：废品率（最小化，[0, 0.4]）**

```python
def calculate_defect_rate(params):
    complexity = params.get("complexity", 0.5)
    base_rate = 0.01 + complexity * 0.04                         # C=0→1%, C=1→5%
    temp_sensitivity = 1.0 + complexity * 1.5                    # 1.0~2.5x
    time_sensitivity = 1.0 + complexity * 1.0                    # 1.0~2.0x
    temp_penalty = (abs(params["heating_temp"] - 620) ** 2) * temp_sensitivity / 140000  # v2.4 平方惩罚
    time_penalty = (abs(params["heating_time"] - 90) ** 2) * time_sensitivity / 270000    # v2.4 平方惩罚
    pressure_penalty = (params["blowing_pressure"] - 0.3) ** 2 / 0.04 * 0.04              # v2.4 平方惩罚
    return min(base_rate + temp_penalty + time_penalty + pressure_penalty, 0.40)
```

- C 越高 → 废品基线升高，对偏离最佳工艺点的敏感性增大（复杂杯型容错性差）

**目标 3：加工能耗（最小化）**

```python
def calculate_energy_consumption(params):
    complexity = params.get("complexity", 0.5)
    complexity_factor = 1.0 + complexity * 0.3                   # C=0→×1.0, C=1→×1.3
    heating = params["heating_temp"] * params["heating_time"] * 0.1 * complexity_factor
    blowing = params["blowing_pressure"] * params["blowing_time"] * 100
    cooling = (60 - params["cooling_rate"]) * 10
    return heating + blowing + cooling
```

- C 越高 → 加热能耗放大系数增大（复杂形状需更高能耗）

**目标 4：耐热安全性（最小化，转换 1−耐热指数）**

```python
def calculate_heat_resistance(params):
    complexity = params.get("complexity", 0.5)
    stress_penalty = complexity * 20
    heat_temp = 120 + params["cooling_rate"] * 0.8 + params["wall_thickness_target"] * 5 - stress_penalty
    return max(0.0, 1 - heat_temp / 200)
```

- C 带来应力集中折减（−20°C at C=1），等效耐热降低

**C 的影响汇总**：

| 目标函数 | C=0 → C=1 变化 | 机制 |
|----------|----------------|------|
| 壁厚均匀度 | 基础标准差 0.12→0.18，均匀化增益降低 | 复杂形状更难均匀 |
| 废品率 | 基线 1%→5%，敏感性 1.0x→2.5x | 容错性差，偏离惩罚放大 |
| 能耗 | 加热系数 ×1.0→×1.3 | 需更高能耗成型 |
| 耐热安全 | 应力折减 0→−20°C | 应力集中降低耐热 |
