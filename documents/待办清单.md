## 🔴 7.16 新增参考要求（6 项）

> 来源：[7.16新增参考要求.md](file:///d:/trae/关于选题/7.16新增参考要求.md)

### 新增核心要求概述

```
AI生成输出特征量 → MOPSO 工艺优化
  轮廓复杂度C      约束不等式更新
  杯身高度H        代入目标函数系数
  口径D            缩小寻优范围
  平均壁厚T        提供唯一匹配对象
        ↓
MOPSO 结果反向反馈
  废品率高 → 提示"降低复杂度"
  能耗大   → 提示"简化造型"
        ↓
迭代闭环：需求 → 生成 → 特征 → 优化 → 评价 → 重生成
```

### 各需求详情

#### [7.16-1] AI生成输出特征量化提取
- [ ] 从 SD 生成图中提取：**轮廓复杂度 C**、**杯身高度 H**、**口径 D**、**平均壁厚 T**
- [ ] 实现方法：OpenCV 轮廓检测 → 计算特征值
- [ ] 特征值传入 MOPSO 目标函数/约束

#### [7.16-2] 特征值动态更新 MOPSO 约束
- [ ] 根据提取的 C/H/D/T 自动调整 MOPSO 搜索边界
- [ ] 高复杂度 C > 0.6 → 自动收紧温度范围 [1000, 1100]
- [ ] 不同杯型（简约/异形）约束各不相同

#### [7.16-3] 目标函数系数引入轮廓复杂度 C
- [ ] 将 C 代入壁厚均匀度公式：增加 C·惩罚项
- [ ] 将 C 代入废品率公式：复杂度越高 → 越容易出废品
- [ ] 现有公式改造：从静态系数 → 动态 C 相关系数

#### [7.16-4] MOPSO 结果反向指导生成迭代
- [ ] 废品率过高 → 系统提示"当前异形轮廓加工难度大，建议降低复杂度"
- [ ] 能耗过大 → 系统提示"当前造型能耗高，建议简化造型"
- [ ] 生成迭代闭环：工艺评价 → 用户调整 prompt → 重新生成

#### [7.16-5] 正向展示效果——设计驱动工艺
- [ ] 页面左侧展示 AI 效果图 + 标注 C/H/D/T 特征值
- [ ] 页面中间显示"工艺参数搜索区间约束说明"
- [ ] 页面右侧运行 MOPSO，输出该造型专属最优工艺参数
- [ ] 不同杯子 → 工艺参数不同 → 证明造型决定工艺

#### [7.16-6] 反向展示效果——工艺约束设计
- [ ] 同一套工艺参数代入简约杯 vs 异形杯 → 对比差异
- [ ] 对比图：两张效果图并排 + 各自四项工艺指标得分
- [ ] 系统弹窗反馈：工艺不达标 → 语义修改建议
